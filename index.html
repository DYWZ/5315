What is Docker

Docker is a tool that uses containers to create, deploy, and run applications more efficiently. A container is a standard software unit that packages code and all its dependencies together, allowing applications to run quickly and reliably from one computing environment to another. Docker container image is a lightweight, stand-alone, executable package that contains code, runtime, system tools, system libraries, and Settings to support the running of applications. With the help of the container, a developer can run an application on any other Linux machine without worrying about the difference between running the device's custom Settings and the device used to write the code. Sources: "What is Docker" - https://opensource.com/resources/what-docker, "What is a Container" - https://www.docker.com/resources/what-container

What is Nginx

As an open source software, NGINX can be used for web services, reverse proxy, caching, load balancing, media streaming, and more. It was initially designed for maximum performance and stability. NGINX features HTTP server functionality, email proxy server capabilities, and reverse proxy and load balancer capabilities for HTTP, TCP, and UDP servers. Source: "What is NGINX?" - https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

A virtual machine is a computer file that is based on a computer architecture that simulates a computer system and provides the functionality of a physical computer, also known as an image. The software in the virtual machine does not have a permanent impact on the computer's hard drive, which provides an ideal environment for creating operating system backups, testing other operating systems, and the like for accessing virus-infected data. Source: "What is a virtual machine?" - https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a U.S. cloud infrastructure provider based in New York City with data centers worldwide. DigitalOcean provides cloud services for developers to help deploy and extend applications running on multiple computers at the same time. Source: "DigitalOcean" - https://en.wikipedia.org/wiki/DigitalOcean
